# smallBeginJava11

## 1. 프로젝트 설명
#### Service architecture (이미지 - 미리캔버스 등 사용)

<br>

#### 프로젝트 기간
기획 : 2021.07~2021.11
개발 : 2022.11 ~ 2022.08

<br>

#### 기술 스택
Front : HTML, CSS, Thymeleaf, Vanila JS
Back : Spring Boot, MySQL(AWS RDS), Ec2(Server)
