# smallBeginning
## 프로젝트 소개
To-do 리스트 형식의 간단한 일정정리 웹 애플리케이션  
달별로 해당 날짜의 To-do 리스트를 확인하고 관리하는 캘린더

<br>
#### 프로젝트 기간
기획: 2022.01 ~ 2022.07
개발: 2022.07 ~ 2022.08
배포: 2022.08 ~ 2022.09

<br>
#### 기술 스택
Front : HTML, CSS, Thymeleaf, Vanila JS
Back : Spring Boot, MySQL(AWS RDS), Ec2(Server)
Dev Tools : Github, Intellij, Putty, Miro
